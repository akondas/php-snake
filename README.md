## Requirements

* PHP >= 7.0.0
* Composer

## Install

Clone from GitHub:

```
git clone git@github.com:itcraftsmanpl/php-snake.git
cd php-snake
composer install --no-dev
```

## Launch

Run simple command in CLI:

```
php snake.php
```

Keys:

```
w - move up
a - move left
s - move down
d - move right
x - exit to console
```
